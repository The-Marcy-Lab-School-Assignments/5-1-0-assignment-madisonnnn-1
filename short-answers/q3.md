# Question
In a code snippet, demonstrate a `User` class with at least one:
- class property
- class method
- instance property
- instance method

# Answer
```js
class User {
  static platforms = []
  friends = []

  constructor(name,age){
    this.name = name
    this.age = age
  }

  static addPlatform(platform) {
    User.platforms.push(platform)
  }

  addFriend(friend){
    this.friends.push(friend)
  }
}
```
